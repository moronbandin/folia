# Senhora do Almortão

**Senhora, Senhora do Almortão**<br>
Senhora do Almortão<br>
Ó minha linda raiana.

**Virai costas** // Virai costas<br>
Virai costas a Castela<br>
**Virai costas a Castela**<br>
Não queirais ser castelhana.

**Senhora** // Senhora<br>
Senhora do Almortão,<br>
Senhora do Almortão<br>
A vossa capela cheira.

Senhora,<br>
Senhora do Almortão<br>
